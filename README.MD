- Chinese (Simplified)
- Chinese (Traditional)
- English (US)
- Korean (~~North Korea~~)
- Korean (South Korea)
- German (Germany)
### Misc Languages
- Na'vi (Pandora)

Supports Game saving and Game opening

## Executable
Windows executable built by Launch4j-maven-plugin
Macos executable built by macosappbundler-maven-plugin

## Building

Clone the directory using
```bash
git clone https://github.com/QIU2014/GoBangGame.git
```
Navigate to cloned directory (cd GoBangGame), execute:
```bash
mvn clean build
```
or, if you don't want all executables
```bash
ant
```
for the jar only
