# GoBangGame
A small chinese game written in java

## THIS GAME REQUIRES JAVA VERSION 17 AND ABOVE

## Changelog

### Now supports language
- Chinese (Simplified)
- Chinese (Traditional)
- English (US)
- Korean (~~North Korea~~)
- Korean (South Korea)

Supports Game saving and Game opening

## Executable
Windows executable built by **Launch4j**

Workspace & IDE using Eclipse

## Building
Open it using Eclipse IDE for Java
